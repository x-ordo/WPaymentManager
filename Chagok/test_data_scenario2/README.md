# 테스트 시나리오 2: 가정폭력 + 경제적 학대

## 시나리오 개요

**원고(아내)**: 이수진 (37세)
**피고(남편)**: 박철민 (39세, 개발팀 과장)
**결혼 기간**: 10년 (2015년 결혼)
**자녀**: 2명 (초등학생)

### 핵심 이슈
1. **경제적 학대**: 월 실수령 389만원 중 생활비로 150만원만 지급
2. **가정폭력**: 언쟁 중 물건 투척으로 인한 상해
3. **정서적 학대**: 지속적인 폭언 및 통제

2. 경제적 학대 - 충분한 수입에도 생활비 미지급
