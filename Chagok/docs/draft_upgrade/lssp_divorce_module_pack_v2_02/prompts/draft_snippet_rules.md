# draft_snippet_rules (v2.02)

## 목표
LLM이 초안 문구를 제안할 때 “허위 단정 / 과장 / 법률자문 오해”를 줄이고,
증거가 없는 문장은 자동으로 “추정/정황” 표시를 하도록 강제한다.

## 규칙
1) **증거 없는 단정 금지**
- EvidenceKeyPoint.strength != strong 인 경우:
  - “~로 확인된다” → “~로 보인다/추정된다/정황이 있다”

2) **날짜/기간은 반드시 필드 기반**
- 사건 날짜는 {{timeline.*}} 값만 사용
- 제척기간/마감은 rules 엔진이 계산한 {{deadline.*}}만 사용

3) **법조항·판례는 ‘판단틀’로만**
- “대법원은 반드시 이긴다” 식 금지
- “판례상 고려요소” 형태로 정리

4) **UI 출력 문구**
- 경고(효력 소멸형): “기한 경과 시 절차를 처음부터 다시 진행해야 할 수 있습니다.”
- 과태료형: “미신고 시 과태료 부과 가능성이 있습니다.”

5) **문단 출력 포맷**
- [사실] 3~5문장
- [법적 평가] 2~3문장
- [입증] Evidence refs 목록(갑1, 갑2 …)
